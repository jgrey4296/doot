================================
Doot: The Documentation
================================

Doot is a TOML based Task Runner similar to `doit`_.


.. _intro:

Introduction
------------

Doot grew out of wanting a task runner with less esoteric cantations than those of programs like make.


Overview
--------

.. code:: bash

   doot list
